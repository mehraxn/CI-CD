# CloudOps IaC & Observability Lab

A junior-friendly, production-style DevOps portfolio lab built around a small Python web application.

The repository will demonstrate containerization, infrastructure as code, configuration management, monitoring, backups, operational documentation, and automated validation.

## Repository structure

- `app/` - Small Python web application used as the lab workload.
- `terraform/` - Cloud infrastructure skeleton and example Terraform configuration.
- `ansible/` - Server configuration skeleton, inventory examples, and playbooks.
- `monitoring/` - Prometheus configuration and Grafana dashboard definitions.
- `nginx/` - Nginx reverse-proxy configuration.
- `scripts/` - Backup, restore, and operational helper scripts.
- `docs/` - Architecture notes and operational runbooks.
- `.github/workflows/` - GitHub Actions validation workflows.

No real cloud credentials or secrets belong in this repository.
