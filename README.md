# CI/CD Learning Repository

This repository combines structured CI/CD learning materials with practical projects. The lessons establish the concepts and vocabulary; the projects provide places to inspect, apply, and document those concepts.

## Start Here

- [Learning materials](./01-learning-materials/README.md)
- [45-day roadmap](./01-learning-materials/45-day-roadmap.md)
- [Practical projects](./Projects/)

Begin with the learning-materials overview, follow the recommended topic order, and use the roadmap to connect each phase to relevant project files. Project implementations remain independent of the learning notes.
